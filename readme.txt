== Description ==
this plugin is for newsletters compaigns, where user can subscribe from your page and you 
can group the users against the lists and you can make compaigns and also from this plugin you can send welcome massege to register user as well as you can design your email template through email editor. you can manage the spam users as well 

= Main Features =

* NEW **responsive Forms Drag & Drop composer**
* **Unlimited Subscribers**
* **Unlimited Emails**
* Wordpress User Registration **seamless integration**
* **Subscribers preferences** to fine-target your campaigns
* Manage Spam Subscribers
* SMTP-Ready 
* Themes Editor
* Html and Text versions of Emails 
* Responsive and easy to use
* **Compatible with Postman, WP Mail SMTP, Easy WP SMTP, Easy SMTP Mail, WP Mail Bank**

== Installation ==

1. Put the plug-in folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Optional: go to the options page and configure the plugin
